# プロダクト要求定義書

## コンセプト
仕事の対応漏れをゼロにする、リマインド付きタスク管理アプリ

## 対象ユーザー
メール・依頼・タスクの対応を忘れてしまうことがある会社員・ビジネスパーソン

## 解決する課題
- タスクの期限を忘れて対応漏れが発生する問題
- 大量のタスクの中で「いつまでに何をすべきか」が把握しにくい問題
- 漏れに気づくのが遅れて、対応が手遅れになる問題

## 差別化機能
- **リマインド機能**：タスクに期限を設定すると、期限前に自動でリマインド通知が届く
- **漏れ通知機能**：期限を過ぎても未完了のタスクがある場合、指定したメールアドレスに自動でアラートメールを送信する

## ユーザーストーリー

- ユーザーとして、タスクに期限（日時）を設定したい。そうすることで、「いつまでに対応すべきか」を一目で確認できる。
- ユーザーとして、期限前にリマインド通知を受け取りたい。そうすることで、対応を忘れずに済む。
- ユーザーとして、期限切れのタスクがある場合に指定メールアドレスへアラートを受け取りたい。そうすることで、対応漏れに早く気づける。
- ユーザーとして、タスクを完了済みにしたい。そうすることで、対応済みの案件を管理できる。
- ユーザーとして、自分のタスクだけを見たい。そうすることで、他の人のタスクと混在せずに管理できる。

## 完成の定義
- 全機能が本番環境（インターネット上）で動いている
- ログインしたユーザーが自分のタスクだけを操作できる
- 期限切れタスクが指定のメールアドレスに自動通知される
